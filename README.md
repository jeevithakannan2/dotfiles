## 🚀 Getting Started

To get a my dots you can run the installation script:
```bash
curl -fsLo /tmp/jeev-installer.sh https://github.com/jeevithakannan2/dotfiles/releases/latest/download/install.sh && sh /tmp/jeev-installer.sh
```
